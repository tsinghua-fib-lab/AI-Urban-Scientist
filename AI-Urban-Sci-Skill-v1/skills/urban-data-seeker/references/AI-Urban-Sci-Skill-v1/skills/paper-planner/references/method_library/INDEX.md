# Lightweight Method Library Index

This library stores reusable planning references for paper design. Each entry links a method specification to a paper-planning pattern: evidence chain, figure architecture, narrative arc, central claim shape, and claim strength.

The library is organized by domain category, with one markdown file per category. Categories are dynamic rather than fixed. Examples might include `causal_inference.md`, `epidemiology.md`, `human_mobility.md`, `climate_impacts.md`, or any new domain that becomes useful.

Use this index first. Match the user's idea by domain, method, outcome, exposure, population, or keyword, then open only the relevant category file(s). If an idea spans multiple domains, check multiple category files.
