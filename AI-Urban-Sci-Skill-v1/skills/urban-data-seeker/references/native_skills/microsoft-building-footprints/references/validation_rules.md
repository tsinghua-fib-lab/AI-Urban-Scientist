# Validation Rules

- SourceSkill output is source-prior evidence only.
- Confirm the official publisher, documentation, access status, and license.
- Verify geography, time range, variables, product/version identifiers, and provenance.
- Restricted or commercial sources require explicit authorization before any probe or fetch.
- Platform tools, when listed, provide mechanics only and cannot decide source fit.
- Downstream resolver, probe, acquisition, and verifier stages remain required.
