# Document Portal Platform Notes

Planning and government document portals often combine:

- a landing page
- a navigable HTML version
- one or more downloadable PDF artifacts

A platform helper can normalize those entrypoints and keep them as not-final hints. The consuming SourceSkill or route runner still owns:

- source-family suitability
- adopted-vs-draft status checks
- attachment and version validation
- downstream resolver/probe/download/verifier decisions

These notes must not be treated as an answer table or known-source substitute.
